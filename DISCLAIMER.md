# Disclaimer

AgentStage is a localhost-oriented, single-user portal for mounting and browsing static pages generated by AI agents.

It is not designed as:

- a multi-tenant hosting platform
- an authenticated publishing system
- a hardened sandbox for untrusted HTML
- an internet-facing production web service

Before exposing it beyond `127.0.0.1`, review and redesign the threat model.

Key operational assumptions:

- mounted pages are trusted by the operator
- source assets may live in arbitrary local workspaces
- the embedded page can execute its own client-side JavaScript
- the default runtime model is local desktop use

Use this project only in environments where these assumptions are acceptable.
