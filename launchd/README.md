# AgentStage LaunchAgent

This directory stores LaunchAgent assets for AgentStage.

- `com.agentstage.daemon.plist.template`: Placeholder template used by `scripts/launchd-install.mjs`.
- `com.agentstage.daemon.plist`: Last rendered plist for this workspace (generated during install).

Per-user installed location:

- `~/Library/LaunchAgents/com.agentstage.daemon.plist`
